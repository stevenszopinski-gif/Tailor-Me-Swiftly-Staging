**TailorMeSwiftly.com** is a high-performance, single-page application designed to help job seekers instantly customize their resumes and cover letters for specific job roles.

## Key Features
- **AI-Powered Tailoring**: Uses Google Gemini to analyze job descriptions and extract relevant keywords.
- **Privacy-First**: Operates directly in your browser with local parsing and generation.
- **Fast PDF Export**: Get professional, perfectly formatted PDFs with 0.5-inch margins.
- **Writing Preferences**: Set custom tailoring instructions (e.g., "British English", "Focus on leadership").
- **Modern UI**: Clean, purple-themed glassmorphism design for a premium feel.

## How to Use
1. **Upload Resume**: Support for PDF, DOCX, and TXT.
2. **Paste Job Link**: Just paste the URL of the job you're interested in.
3. **Generate**: Get your tailored set instantly.

## Tech Stack
- HTML5 / Vanilla CSS
- JavaScript
- Google Gemini API (gemini-3-flash-preview)
- pdf.js / Mammoth.js / html2pdf.js
